\section{Simple theories}
\vspace{10.5cm}

In this chapter $T$ is always a complete theory in a first-order language $L$. We work in a huge $\kappa$-saturated
model $M$ where $\kappa$ is an arbitrary chosen cardinal number greater than all other sets and models of the 
discource.

All models $A, B, C, \dots$ are elementary submodels of $M$, all sets $X, Y, Z,\dots$ are subsets of the domain of 
$M$. All types $p, q, r, \dots$ are consistent with $T$.

A type $p(\bar{x})$ divides over a set $X$ with respect to a natural number $k$, if there are a formula $\phi(\bar{x}, \bar{c})$ with
parameters $\bar{c}$ and a sequence $(\bar{c}_i: i < \omega)$ such that

\begin{enumerate}[label={(\roman*)}]
    \item 
    $p\entails\phi(\bar{x}, \bar{c})$
    \item 
    $tp(\bar{c}/X) = tp(\bar{c}_i/X)$
    \item 
    any finite subset of size $k$ of $\{\phi(\bar{x}, \bar{c})_i\}$ is inconsistent
\end{enumerate}
